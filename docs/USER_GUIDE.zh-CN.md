## 入门

### 访问应用

1. 打开您的网页浏览器
2. 导航到应用 URL：
   - 开发环境：`http://localhost:5173/view/`
   - 生产环境：您配置的域名 + `/view/`

### 了解 TSFile 模型

TSFile 查看器支持两种数据模型：

1. **树模型**：传统的基于路径的层次模型
   - 设备路径如 `root.sensor1.temperature`
   - 适用于 IoT 传感器数据

2. **表模型**：基于关系表的模型（TSFile V4）
   - 包含 TAG 列（标识符）和 FIELD 列（测量值）的表
   - 适用于结构化时间序列数据

应用会自动检测您的 TSFile 使用的模型类型。

## 文件选择

### 浏览服务器文件

1. 在主页上，您会在左侧看到文件树
2. 点击文件夹图标展开目录
3. 浏览目录结构
4. 点击 `.tsfile` 文件查看其元数据

**注意**：只能访问服务器白名单中配置的目录。

### 上传文件

1. 在主页上，找到右侧的"上传 TSFile"卡片
2. 可以：
   - 将 `.tsfile` 文件拖放到上传区域
   - 点击上传区域浏览并选择文件
3. 等待上传完成（进度条显示状态）
4. 应用将自动导航到元数据视图

**上传要求：**

- 文件必须具有 `.tsfile` 扩展名
- 最大文件大小：100MB（可由管理员配置）

### 最近访问的文件

"最近访问"部分显示您最近访问的文件：

- 点击任何文件卡片快速返回其元数据视图
- 文件存储在浏览器的本地存储中
- 最多保留 10 个最近文件

## 查看元数据

选择文件后，您将看到包含 TSFile 完整信息的元数据页面。

### 基本信息

顶部显示的卡片包含：

- **版本**：TSFile 格式版本
- **时间范围**：数据的开始和结束时间戳
- **设备数量**：文件中的设备数
- **测点数量**：测量点数量
- **设备数量**：RowGroup 数量
- **数据块数量**：Chunk 数量

### 模型切换

使用"视图模式"切换按钮在以下模式间切换：

- **树模型**：在平面表格中显示测点
- **表模型**：显示包含 TAG 和 FIELD 列的表模式

### 树模型视图

**测点表格：**

- 列出所有测点及其属性
- 列：名称、数据类型、编码、压缩
- 点击列标题排序
- 使用搜索框按名称筛选

**设备表格：**

- 按设备和时间显示数据组织
- 列：索引、设备、开始时间、结束时间、数据块数量
- 点击行展开查看详细信息

**数据块表格：**

- 显示数据存储的物理组织
- 列：测点、偏移量、大小、压缩率
- 帮助理解文件结构和性能特征

### 表模型视图

**表模式：**

- 显示表名和列定义
- **TAG 列**：用于标识和分组的列
- **FIELD 列**：包含实际测量值的列
- 每列显示：名称、数据类型、编码、压缩

## 预览数据

从元数据页面，点击"数据预览"按钮查看实际数据。

### 应用筛选条件

使用左侧的筛选面板缩小数据范围：

**时间范围：**

- 设置开始和结束时间
- 使用日期时间选择器或手动输入
- 留空以包含所有时间

**设备：**

- 输入设备名称（逗号分隔）
- 示例：`root.sensor1, root.sensor2`
- 留空以包含所有设备

**测点：**

- 输入测点名称（逗号分隔）
- 示例：`temperature, humidity`
- 留空以包含所有测点

**数值范围：**

- 设置最小值和/或最大值
- 仅显示在此范围内的值
- 适用于所有数值列

**分页：**

- **每页条数**：每页显示的行数（1-1000）
- **偏移量**：跳过的行数（用于分页）

点击"应用筛选"查看结果。

### 浏览数据

数据表格显示：

- **时间戳**：每行的时间戳
- **设备**：设备标识符
- **测点列**：每个测点一列，显示其值

**表格功能：**

- 点击列标题排序
- 使用底部的分页控件导航
- 查看总行数和可用数据指示器

### 导出数据

将筛选后的数据导出为不同格式：

**CSV 导出：**

- 点击"导出 CSV"按钮
- 下载包含标题行的逗号分隔文件
- 可在 Excel 或其他电子表格软件中打开

**JSON 导出：**

- 点击"导出 JSON"按钮
- 下载结构化 JSON 数组
- 适用于编程处理

**注意**：导出包含当前筛选条件下的所有数据，而不仅仅是当前页面。

## 可视化数据

从元数据或数据预览页面，点击"图表可视化"查看交互式图表。

### 选择测点

1. 使用"选择测点"下拉菜单
2. 选择一个或多个要可视化的测点
3. 选择的测点将显示为不同颜色的线条

### 应用筛选条件

使用与数据预览相同的筛选面板：

- 时间范围
- 设备
- 测点
- 数值范围

### 聚合选项

对于大型数据集，应用聚合以提高性能：

**聚合类型：**

- **最小值**：时间窗口内的最小值
- **最大值**：时间窗口内的最大值
- **平均值**：时间窗口内的平均值
- **计数**：时间窗口内的数据点数

**时间窗口：**

- 设置聚合间隔（毫秒）
- 示例：60000 = 1 分钟窗口

### 与图表交互

**缩放和平移：**

- 使用鼠标滚轮缩放
- 拖动图表平移
- 使用底部的滑块选择时间范围

**工具箱：**

- **数据缩放**：框选区域放大
- **还原**：重置视图
- **保存为图片**：下载图表

**悬停提示：**

- 将鼠标悬停在数据点上查看精确值
- 显示时间戳和所有序列的值

**钻取：**

- 点击数据点查看详细信息
- 弹出对话框显示该时间点的所有测量值

### 导出图表

将可视化保存为图像：

**PNG 导出：**

- 点击"导出 PNG"
- 下载高分辨率光栅图像
- 适用于文档和演示

**SVG 导出：**

- 点击"导出 SVG"
- 下载可缩放矢量图形
- 适用于出版物和进一步编辑

### 降采样

对于超过 1000 个数据点的大型数据集：

- 应用自动应用 LTTB（最大三角形三桶）降采样
- 保留视觉形状同时减少点数
- 图表下方显示降采样指示器
- 原始数据点数显示以供参考

## 提示和最佳实践

### 性能优化

1. **使用时间范围筛选**：
   - 始终指定合理的时间范围
   - 避免一次加载整个文件

2. **限制测点**：
   - 仅选择需要的测点
   - 过多序列会降低图表性能

3. **使用聚合**：
   - 对于长时间范围，应用聚合
   - 减少数据点同时保留趋势

4. **分页数据**：
   - 使用合理的每页条数（100-500）
   - 对于大型结果集使用偏移量

### 数据探索

1. **从元数据开始**：
   - 查看时间范围和可用测点
   - 了解数据结构

2. **使用筛选条件**：
   - 从宽筛选条件开始
   - 根据需要逐步细化

3. **比较测点**：
   - 在图表上叠加多个测点
   - 识别相关性和模式

4. **导出以供分析**：
   - 将筛选后的数据导出到外部工具
   - 使用 CSV 进行电子表格分析
   - 使用 JSON 进行编程处理

### 文件管理

1. **组织文件**：
   - 使用描述性文件名
   - 按项目或时间段分组

2. **使用最近访问**：
   - 快速访问常用文件
   - 无需每次浏览

3. **验证上传**：
   - 上传后检查元数据
   - 确认时间范围和测点

## 故障排除

### 常见问题

**文件未显示在树中**

- 检查文件是否在白名单目录中
- 验证文件扩展名为 `.tsfile`
- 刷新浏览器

**上传失败**

- 检查文件大小（<100MB）
- 验证文件扩展名
- 检查网络连接
- 查看浏览器控制台错误

**元数据加载缓慢**

- 大文件可能需要时间解析
- 等待加载完成
- 检查服务器日志错误

**数据预览为空**

- 验证筛选条件
- 检查时间范围是否与数据匹配
- 尝试移除所有筛选条件

**图表不显示**

- 确保选择了测点
- 检查筛选条件是否返回数据
- 尝试更宽的时间范围

**导出失败**

- 检查浏览器弹出窗口阻止程序
- 验证有足够的磁盘空间
- 尝试较小的数据集

### 错误消息

**"文件未找到"**

- 文件可能已被删除或移动
- 检查文件路径
- 尝试重新上传

**"无效的 TSFile"**

- 文件可能已损坏
- 验证文件是有效的 TSFile
- 尝试不同的文件

**"访问被拒绝"**

- 文件不在白名单目录中
- 联系管理员添加目录

**"请求超时"**

- 查询时间过长
- 使用更具体的筛选条件
- 减少时间范围

### 获取帮助

如果您遇到问题：

1. 检查浏览器控制台错误
2. 查看服务器日志
3. 验证配置设置
4. 查阅 API 文档
5. 在 GitHub 上提交问题

### 浏览器兼容性

推荐的浏览器：

- Chrome/Edge 90+
- Firefox 88+
- Safari 14+

功能需求：

- JavaScript 已启用
- Cookie/本地存储已启用
- 现代 ES6+ 支持

## 高级功能

### 键盘快捷键

- `Ctrl/Cmd + F`：聚焦搜索
- `Esc`：关闭对话框
- `←/→`：在图表中导航

### URL 参数

直接链接到特定视图：

- `/view/` - 文件选择
- `/view/meta/:fileId` - 元数据
- `/view/data/:fileId` - 数据预览
- `/view/chart/:fileId` - 图表

### 本地存储

应用在浏览器中存储：

- 最近访问的文件（最多 10 个）
- 语言偏好
- 筛选条件偏好（即将推出）

清除本地存储：

- 浏览器设置 > 清除浏览数据
- 或使用浏览器开发工具
