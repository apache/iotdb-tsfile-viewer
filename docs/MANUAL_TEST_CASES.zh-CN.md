# TSFile Viewer 手动测试用例

## 测试概述

本文档提供完整的手动测试用例，覆盖前端UI重构后的所有功能。每个测试用例包含详细的步骤、预期结果和验证要点。

## 测试环境准备

### 前置条件
1. 后端服务运行在 `http://localhost:8080`
2. 前端服务运行在 `http://localhost:5173`
3. 至少有一个可访问的 `.tsfile` 文件
4. 浏览器：Chrome 90+, Firefox 88+, 或 Safari 14+

### 测试数据准备
- 准备一个包含多个设备和测点的TSFile
- 时间范围覆盖至少24小时
- 包含数值类型的测点（用于排序测试）
