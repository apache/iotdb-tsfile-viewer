# TSFile Viewer 用户手册 - 新功能补充

本文档补充说明前端UI重构后新增的功能和改进的使用方式。完整的基础功能请参考 [USER_GUIDE.zh-CN.md](USER_GUIDE.zh-CN.md)。

## 新增功能概览

### 1. 智能数据筛选（FilterPanel增强）

#### 1.1 快捷时间范围选择

**位置**：数据预览页面 → 筛选面板

**功能**：一键选择常用时间范围，无需手动输入开始/结束时间

**使用方法**：
1. 打开数据预览页面
2. 在筛选面板中找到"时间范围"部分
3. 点击以下任一快捷按钮：
   - **最近1小时**：查看最近1小时的数据
   - **最近6小时**：查看最近6小时的数据
   - **最近24小时**：查看最近24小时（1天）的数据
   - **最近7天**：查看最近一周的数据
   - **最近30天**：查看最近一个月的数据

4. 系统自动填充开始时间和结束时间
5. 点击"应用筛选"查看结果

**提示**：
- 快捷按钮会高亮显示当前选中的范围
- 可以在快捷选择后手动微调时间
- 时间基于当前系统时间计算

#### 1.2 高级过滤 - 值范围筛选

**位置**：数据预览页面 → 筛选面板 → "显示高级筛选"按钮

**功能**：按数值范围过滤数据，只显示符合条件的记录

**使用方法**：
1. 在筛选面板中点击"显示高级筛选"按钮
2. 展开的区域中找到"值范围"设置
3. 设置筛选条件：
   - **最小值**：只显示大于等于此值的数据
   - **最大值**：只显示小于等于此值的数据
   - 可以只设置一个，或同时设置两个

4. 点击"应用筛选"

**使用场景**：
- 过滤异常值（如温度 > 100°C）
- 查找特定范围内的数据（如湿度 40-60%）
- 排除无效数据（如排除值为 -999 的错误数据）

**示例**：
```
场景：查找温度在 20-30°C 之间的数据
设置：最小值 = 20，最大值 = 30
结果：只显示温度在此范围内的所有记录
```

#### 1.3 响应式筛选面板

**功能**：筛选面板自动适配不同屏幕尺寸

**表现**：
- **大屏幕（≥1024px）**：3列网格布局
- **中等屏幕（768-1023px）**：2列网格布局
- **小屏幕（<768px）**：1列堆叠布局

**优势**：移动设备和平板上也能轻松使用筛选功能

### 2. 数据表格增强（DataTable新功能）

#### 2.1 列排序

**位置**：数据预览页面 → 数据表格

**功能**：点击列头对数据进行排序

**使用方法**：
1. 在数据表格中找到任意列头
2. 点击列头进行排序：
   - **第一次点击**：升序排序（小到大、A到Z）
   - **再次点击**：降序排序（大到小、Z到A）

3. 当前排序列会显示箭头指示器：
   - ↑ 表示升序
   - ↓ 表示降序

**支持的列**：
- **时间戳列**：按时间顺序排序
- **设备列**：按设备名称字母顺序排序
- **测点列**：按数值大小排序（数字类型）或字母顺序（字符串类型）

**智能排序**：
- 数字按数值大小排序
- 字符串按字母顺序排序
- 系统自动识别数据类型

**注意**：
- 排序仅应用于当前页面的数据
- 更改筛选条件会重置排序
- 排序不影响服务器数据，仅在浏览器中处理

#### 2.2 响应式分页控件

**功能**：分页控件自动适配小屏幕

**表现**：
- **大屏幕**：横向排列（信息在左，按钮在右）
- **小屏幕**：纵向堆叠（信息和按钮分两行）

**优势**：移动设备上更易操作分页按钮

### 3. 图表可视化增强（ChartPanel新功能）

#### 3.1 暗黑模式自适应

**位置**：图表可视化页面

**功能**：图表自动跟随系统/应用暗黑模式切换主题

**使用方法**：
1. 点击页面右上角的暗黑模式切换按钮
2. 图表自动更新为暗黑主题：
   - 背景色变为深色
   - 文字颜色变为浅色
   - 图表线条颜色优化以提高可读性

3. 导出的图片也会使用对应主题的颜色

**优势**：
- 夜间使用更舒适
- 节省显示器功耗（OLED屏幕）
- 演示时可选择合适的主题

#### 3.2 刷新按钮

**位置**：图表可视化页面 → 图表卡片右上角

**功能**：一键重新加载图表数据，无需重新配置筛选条件

**使用方法**：
1. 配置好筛选条件并点击"应用"生成图表
2. 如需更新数据（如数据源已更新），点击刷新按钮（循环箭头图标）
3. 系统使用当前筛选条件重新查询数据
4. 图表自动更新显示最新数据

**使用场景**：
- 数据文件已更新，需要查看最新数据
- 怀疑数据有缓存，需要强制刷新
- 图表显示异常，需要重新加载

#### 3.3 改进的图表工具栏

**新增/改进**：
- 所有按钮添加了图标，更易识别
- 刷新按钮（新增）
- 导出PNG按钮（带图标）
- 导出SVG按钮（带图标）
- 数据为空时按钮自动禁用

### 4. 图表配置优化（ChartVisualizationView改进）

#### 4.1 统一配置面板

**改进前**：7个独立的卡片，界面杂乱

**改进后**：单一的统一配置卡片，按逻辑分组

**新的分组结构**：

**1. 测点选择**
- 输入测点名称（逗号分隔）
- 帮助文本：提示输入格式

**2. 时间范围**
- 快捷时间范围按钮（同数据预览）
- 自定义开始/结束时间
- 响应式布局（2列→1列）

**3. 聚合设置**
- 聚合类型选择（无、最小值、最大值、平均值、计数）
- 时间窗口大小（仅在选择聚合时显示）
- 帮助文本：解释参数含义

**4. 性能设置**
- 最大点数设置
- 帮助文本：说明降采样机制

**5. 操作按钮**
- 应用按钮（带loading状态）
- 重置按钮（一键清空所有设置）

**优势**：
- 逻辑清晰，易于理解
- 减少视觉噪音
- 响应式设计，移动友好

#### 4.2 重置功能

**位置**：图表配置面板底部

**功能**：一键清空所有配置，恢复默认值

**使用方法**：
1. 在图表配置面板中点击"重置"按钮
2. 所有设置恢复为默认值：
   - 测点选择：清空
   - 时间范围：清空
   - 聚合类型：无
   - 最大点数：1000

3. 图表清空，显示空状态

**使用场景**：
- 开始新的数据探索
- 配置错误，快速重置
- 清除所有筛选条件

### 5. 统一的空状态设计

**位置**：所有数据显示区域

**改进**：统一的空状态样式，包含图标和提示文字

**表现**：
- **图表空状态**：显示图表图标 + "暂无数据显示，请选择测点并应用筛选条件"
- **表格空状态**：显示收件箱图标 + "未找到数据"

**优势**：

- 视觉上更友好

### 6. 加载状态优化

## 使用技巧与最佳实践

### 快速数据探索流程

1. **选择文件** → 查看元数据了解数据概况
2. **数据预览** → 使用快捷时间范围快速查看最近数据
3. **筛选数据** → 展开高级筛选，设置值范围过滤异常值
4. **排序查看** → 点击列头排序，找出最大/最小值
5. **图表可视化** → 使用相同筛选条件生成图表
6. **导出结果** → 导出筛选后的数据或图表

### 性能优化建议

1. **使用快捷时间范围**：避免查询过大的时间跨度
2. **利用值范围筛选**：提前过滤掉不需要的数据
3. **合理使用聚合**：长时间范围使用聚合功能
4. **适当的最大点数**：图表点数1000-5000为最佳

### 移动设备使用

1. **响应式布局**：所有功能自动适配小屏幕
2. **触摸操作**：所有按钮触摸目标足够大
3. **滑动查看**：表格支持水平滑动查看所有列
4. **快捷按钮**：在移动设备上更易点击

## 故障排除

### 快捷时间范围不工作
- **问题**：点击快捷按钮没有反应
- **解决**：检查浏览器控制台是否有JavaScript错误，刷新页面重试

### 列排序显示异常
- **问题**：排序后数据顺序不对
- **解决**：确认数据类型，数字和字符串排序规则不同

### 暗黑模式图表不切换
- **问题**：切换暗黑模式后图表主题不变
- **解决**：点击图表的刷新按钮，强制重新初始化图表

### 高级筛选找不到
- **问题**：看不到值范围筛选选项
- **解决**：点击筛选面板中的"显示高级筛选"按钮展开
